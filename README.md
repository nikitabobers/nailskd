# nailskd

FullStack reservation app.

## Built With

React, Typescript, Graphql, NodeJs, Exress

## Setting up Dev

Install application:

```shell
git clone https://github.com/nikitabobers/nailskd
cd nailskd/server
yarn install
```

```shell
cd nailskd/client
yarn install
```

Install both server and client parts.

## Database

Heroku + Postgres
